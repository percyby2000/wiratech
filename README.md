## ¿Cómo ejecutar este proyecto?

1. Clona el repositorio:
   ```bash
   git clone <url-del-repo>
   cd wiratech
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
4. Abre tu navegador en [http://localhost:3000](http://localhost:3000)
